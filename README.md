# Git Differ

This project is a simple web application that allows you to view differences between git commits.

## Installation

To install and run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/git-differ.git
    cd git-differ
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the server:**
    ```bash
    npm start
    ```
    The application will be accessible at `http://localhost:3000`.

## How to Use

-   **Search Commits:** Press `Cmd+K` (or `Ctrl+K` on Windows/Linux) to open the search bar and search for specific commits.
-   **View Diffs:** Click on a commit to view the detailed differences.

## Tech Stack

-   **Frontend:** HTML, CSS, JavaScript (public/app.js)
-   **Backend:** Node.js (server.js)
-   **Package Management:** npm
